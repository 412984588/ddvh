# Scaffold Skill

Create new components or features in the Voice Hub monorepo.

## Usage

When you need to add a new feature or component to Voice Hub, this skill helps scaffold the necessary files and structure.

## Examples

```
"scaffold a new audio processor for the audio-engine package"
"scaffold a new provider adapter for XYZ service"
```

## Implementation

The skill will:

1. Identify the target package based on the request
2. Create necessary files (source, tests, types)
3. Update exports and dependencies as needed
4. Follow existing code patterns and conventions
