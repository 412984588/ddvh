# Doctor Skill

Run diagnostics on Voice Hub setup and configuration.

## Usage

Use this skill to check the health of your Voice Hub installation.

## Examples

```
"run voice hub doctor"
"check if voice hub is properly configured"
```

## Implementation

The doctor will:

1. Check environment variables are set
2. Verify all dependencies are installed
3. Validate configuration files
4. Check build status
5. Test service connections (optional)
6. Provide recommendations for any issues found
