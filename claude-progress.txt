```
discord-doubao-voice-hub/
├── apps/
│   └── bridge-daemon/          # 主守护进程
├── packages/
│   ├── core-runtime/           # 核心编排逻辑
│   ├── audio-engine/           # 音频处理引擎
│   ├── provider-volcengine-omni/    # 豆包 Omni WebSocket 客户端
│   ├── memory-bank/            # SQLite 历史存储
│   ├── backend-dispatcher/     # 后端任务分发 + webhook
│   ├── shared-config/          # 共享配置和类型
│   ├── openclaw-plugin/        # OpenClaw 插件壳
│   ├── claude-mcp-server/      # MCP Server
│   └── claude-marketplace/     # Claude Code 本地 marketplace
├── scripts/                    # 安装和工具脚本
├── docs/                       # 文档
└── 根配置文件
```

**Phase 3: Provider 适配器**

- types.ts - 类型定义
- event-parser.ts - 事件解析

**Phase 4: Memory Bank**
- schema.sql - 数据库表结构
- DatabaseManager - SQLite 连接管理
- PitfallQueries/PatternQueries - 历史查询
- seed.ts - 示例数据

**Phase 5: Backend Dispatcher**
- BackendClient - HTTP 异步分发
- WebhookServer - Fastify webhook 接收器
- SignatureVerifier - 签名验证 (HMAC SHA256)

**Phase 6: Core Runtime**
- SessionRegistry - 活跃会话追踪
- ActiveConversationStore - 对话历史
- IntentInterceptor - 意图识别和路由
- ResultAnnouncer - 结果播报
- ToolDispatcher - 工具调用分发
- RuntimeBootstrap - 主协调器

**Phase 7: Bridge Daemon**
- DiscordBot - Discord bot 封装

- index.ts - 主入口 (支持 dotenv)

**Phase 8: 插件集成**
