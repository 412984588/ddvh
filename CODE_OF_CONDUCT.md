# Code of Conduct / 行为准则

This project adopts the Contributor Covenant Code of Conduct 2.1.
本项目采用 Contributor Covenant 2.1 行为准则。

Full text:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Our Standards / 行为标准

- Be respectful and inclusive.
- Focus on technical facts and constructive collaboration.
- Avoid harassment, personal attacks, and discrimination.

- 尊重并包容他人。
- 以技术事实和建设性协作为基础。
- 禁止骚扰、人身攻击和歧视。

## Enforcement / 执行

Project maintainers are responsible for clarifying and enforcing this policy.
项目维护者负责解释并执行本准则。

For serious incidents, use private security reporting channels when appropriate.
涉及严重事件时，优先使用私密渠道上报。
